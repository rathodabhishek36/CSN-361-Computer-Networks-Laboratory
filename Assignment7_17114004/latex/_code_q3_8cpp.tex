Builds Huffman Tree and decode given input text. 

count frequency of appearance of each character and store it in a map

Create a priority queue to store live nodes of Huffman tree;

Create a leaf node for each character and add it to the priority queue.

do till there is more than one node in the queue

Remove the two nodes of highest priority (lowest frequency) from the queue

Create a new internal node with these two nodes as children and with frequency equal to the sum of the two nodes\textquotesingle{} frequencies. Add the new node to the priority queue.

root stores pointer to root of Huffman Tree
